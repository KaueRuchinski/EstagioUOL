## Conceitos do JMeter:

O Apache JMeter é uma ferramenta usada principalmente para testar o desempenho e medir a carga de aplicativos web

O JMeter serve para diversas finalidades no contexto de testes de software, entre eles:

- Teste de Carga
- Teste de Estresse
- Teste de Pico
etc...
